# v1.0.0-beta

Initial Android beta for Flight Tickets Price Tracker.

## Included

- Native Android search screen.
- Mom-priority CMN to YUL round-trip defaults for August 2026 departure and a February 2027 return under six months.
- Direct-flight-only default filter.
- Air Canada ranking preference when close in price because of the two-checked-bag need, while still allowing Royal Air Maroc to rank well when considerably cheaper.
- Offline fare quote generator.
- Local target-price alert persistence.
- Unit-tested domain logic.

## APK

The release APK is expected at `app/build/outputs/apk/release/app-release-unsigned.apk` after running `ANDROID_HOME=$PWD/.android-sdk gradle assembleRelease` in an environment with Android Gradle Plugin and SDK access.
