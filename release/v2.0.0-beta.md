# Live Flight Price Tracker v2.0.0-beta

This release replaces the offline simulator with real provider integration.

## Real data

- Amadeus Flight Offers Search
- provider carrier names and IATA codes
- actual flight numbers and itinerary segments
- departure and arrival timestamps
- stops and total duration
- baggage metadata when supplied
- provider currency and total fare
- verification timestamp

## Tracking

- saved target prices
- provider-price history
- approximately six-hour background checks
- Android notifications when a provider fare reaches the target

## Required configuration

The APK deliberately contains no API key. Configure either:

1. a deployed HTTPS backend from the included `server/` directory, or
2. private-device Amadeus credentials entered inside the app.

Production live inventory requires Amadeus production credentials. The test environment is clearly labeled and contains limited provider test data.

## Integrity guarantee

There is no fake-price fallback. Missing credentials, provider errors, quotas, or connectivity problems are shown as errors instead of generated tickets.
