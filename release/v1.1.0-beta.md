# v1.1.0-beta

This beta repairs the original prototype and turns it into a coherent local product demo.

## Highlights

- Validated airport, route, date, passenger, and target inputs.
- Multiple local price alerts with migration from the original storage format.
- Manual alert evaluation and clearer fare cards.
- Honest offline-demo labeling and production integration guidance.
- Reliable GitHub Actions verification and an installable beta APK artifact.

The included prices remain simulated and are not live airline fares.
