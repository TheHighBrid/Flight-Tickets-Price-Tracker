# Flight Tickets Price Tracker v1.1.1-beta

This beta repairs and consolidates the Android build and release pipeline.

## Included

- Android Gradle Plugin 9.3.0 paired with its required Gradle 9.5.0 toolchain.
- JDK 17, Android SDK 35, and Build Tools 36.0.0 setup in CI.
- Core Java smoke tests, unit tests, Android lint, and APK compilation.
- Removal of duplicate and invalid workflows that caused repeated failures.
- Installable debug-signed APK published as a GitHub prerelease asset.
- SHA-256 checksum published beside the APK.

## Product status

The application remains an offline product beta. Fare values are deterministic demo data and are not live airline or travel-agency inventory.
